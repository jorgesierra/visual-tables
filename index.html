Visual Tables

jQuery Bootstrap HTML Tables Visualization Plugin

Download this project as a .zip file Download this project as a tar.gz file

Options

  • Type: bar | circle | disabled
  • Filter Operator: none | = | < | > | <= | >=
  • Filter Value Type: numeric | percentage

Usage

$(".visualized").visualizeTable(
    {type: 'bar',filters: [{'operator':'<','value':'50%'}]}
);

The example above displays bar type filtering out values smaller than 50% (half of the biggest value).
